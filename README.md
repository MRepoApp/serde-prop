# serde-prop
Rust library for serializing and deserializing `prop`

## usage
```toml
[dependencies.serde-prop]
git = "https://github.com/MRepoApp/serde-prop.git"
```

## usage (no-std)
```toml
[dependencies.serde-prop]
git = "https://github.com/MRepoApp/serde-prop.git"
default-features = false
features = ["alloc"]
```
